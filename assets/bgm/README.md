# assets/bgm — BGM 素材のライセンス台帳

このディレクトリに置かれる音声ファイル (*.mp3 等) は **本リポジトリのバージョン管理対象外**です（`.gitignore` で除外）。ここに置くファイルは以下のいずれかを満たす必要があります。

- パブリックドメイン / CC0
- CC-BY, CC-BY-SA 等で、本プロジェクトの配布形態と両立するもの
- 作者から再配布許諾を明示的に得たもの
- 自作 / コミッション委託で権利がプロジェクトに帰属するもの
- AI音楽生成サービスの利用規約で商用再配布が許諾されたアウトプット

## 出典記録の書式

新しい素材を追加したら、以下のテーブルに必ず一行追加してください。出典不明・ライセンス不明のものは**置かない**こと。

| ファイル名 | 曲名 / 作者 | 出典 URL | ライセンス | 備考 |
|---|---|---|---|---|
| _(example)_ 101.mp3 | _(example)_ Sample Track / John Doe | _(example)_ https://... | _(example)_ CC-BY 4.0 | _(example)_ クレジット要表示 |

## 配布について

本プロジェクト自体はエンジン部分のオープンソース公開のみ。音素材を含むビルド成果物の公開配布は行いません。素材は各自の環境でローカルに配置する前提です。
