# assets/se — 効果音素材のライセンス台帳

このディレクトリに置かれる音声ファイル (*.wav 等) は **本リポジトリのバージョン管理対象外**です（`.gitignore` で除外）。ここに置くファイルは以下のいずれかを満たす必要があります。

- パブリックドメイン / CC0
- CC-BY, CC-BY-SA 等で、本プロジェクトの配布形態と両立するもの
- 作者から再配布許諾を明示的に得たもの
- 自作 / コミッション委託で権利がプロジェクトに帰属するもの
- 効果音生成サービスの利用規約で商用再配布が許諾されたアウトプット

## 出典記録の書式

新しい素材を追加したら、以下のテーブルに必ず一行追加してください。出典不明・ライセンス不明のものは**置かない**こと。

| ファイル名 | 音源 / 作者 | 出典 URL | ライセンス | 備考 |
|---|---|---|---|---|
| _(example)_ 100.wav | _(example)_ Menu Click / John Doe | _(example)_ https://freesound.org/... | _(example)_ CC0 | _(example)_ クレジット不要 |

## 配布について

本プロジェクト自体はエンジン部分のオープンソース公開のみ。素材を含むビルド成果物の公開配布は行いません。素材は各自の環境でローカルに配置する前提です。
